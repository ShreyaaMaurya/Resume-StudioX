## 📞 NEED HELP?

### Most Common Questions:

**Q: How do I start?**
A: `npm run dev` then open `http://localhost:5000/Resume.html`

**Q: How do I download my resume?**
A: Click "📄 Download as PDF" button in the left sidebar

**Q: Where are my downloads?**
A: Check your Downloads folder (usually `C:\Users\[YourName]\Downloads`)

**Q: What if download doesn't work?**
A: Click "🖨️ Print" button, then select "Save as PDF"

**Q: Can I create cover letters too?**
A: Yes! Open `http://localhost:5000/cover-letter.html`

**Q: Is it mobile friendly?**
A: Yes, it works on phones and tablets too

**Q: Can I use multiple templates?**
A: Yes, select different templates in section 3

**Q: Is the backend working?**
A: Yes, server runs on port 5000 with all features
