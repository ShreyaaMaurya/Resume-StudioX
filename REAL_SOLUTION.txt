🔧 HOW TO SOLVE THIS

STEP 1: Run Diagnostic (tells you exactly what's wrong)
────────────────────────────────────────────────
npm run dev
node diagnostic-image-upload.js

Watch the output - it will show:
  ✅ Gemini API Key: working/not working
  ✅ Gemini API Connection: working/not working  
  ✅ Gemini Multimodal Support: working/not working
  + Specific error messages

STEP 2: Based on Errors - Fix It
────────────────────────────────────────────────

IF: "PERMISSION_DENIED" or "not enabled"
→ Enable Vision API:
  1. https://console.cloud.google.com/apis/library
  2. Search: "Generative Language API"
  3. Click ENABLE

IF: "RESOURCE_EXHAUSTED" or "QUOTA"
→ Wait a few minutes and try again
→ Check your API quota at Google Cloud Console

IF: "Invalid API key"
→ Regenerate key:
  1. https://makersuite.google.com/app/apikeys
  2. Delete old key
  3. Create new key
  4. Update .env
  5. Restart: npm run dev

STEP 3: Test Again
────────────────────────────────────────────────
npm run dev
Upload an image
Check server logs - should see one of:
  ✅ "Gemini multimodal succeeded"
  ✅ "Text-only Gemini succeeded"
  ❌ "Failed" - shows specific error
