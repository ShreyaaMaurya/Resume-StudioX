## 🎯 HOW TO USE RESUME BUILDER

### 1. Open Resume Builder

```
http://localhost:5000/Resume.html
```

### 2. Look at the Left Sidebar

You'll see 5 sections:

```
1️⃣ ✨ AI BLUEPRINT ENGINE
2️⃣ ⬇️ EXPORT & DOWNLOAD
3️⃣ 01 / LAYOUT & IDENTITY
4️⃣ 02 / MANUAL COMPONENT MODULES
5️⃣ 03 / ACTIVE FIELDS PANEL
```

### 3. Configure Your Resume

**Section 1: AI BLUEPRINT ENGINE**

- Click the "🤖 Build Layout via AI" button to auto-generate structure

**Section 3: LAYOUT & IDENTITY**

- Select your template
- Enter your name
- Enter your professional role
- Add avatar URL (optional)
- Add contact information

**Section 4: MANUAL COMPONENT MODULES**
Click any button to add that section:

- 📝 **Summary** - Overview of your experience
- 💼 **Experience** - Work history
- 🚀 **Projects** - Showcase projects
- 🛠️ **Skills** - List your skills
- 🎓 **Education** - School/Degree info
- 🌐 **Languages** - Languages you speak
- 📜 **Certificates** - Certifications
- 🏆 **Awards** - Awards and achievements

### 4. Edit Your Content

When you click a section button (like 📝 Summary):

- New fields appear in Section 5: "03 / ACTIVE FIELDS PANEL"
- Edit the content
- Live preview on the right updates immediately!

### 5. Download Your Resume

Look for **Section 2: EXPORT & DOWNLOAD**

You have two options:

- 📄 **Download as PDF** - Creates a PDF file that downloads instantly
- 🖨️ **Print Resume** - Opens your browser's print dialog

### 6. Perfect Resume!

Your resume is now ready to share! 📄✨

## 🆘 IF SOMETHING DOESN'T WORK

### Server Won't Start

```bash
# Make sure you're in the right folder
cd c:\Users\admin\OneDrive\Desktop\ResumeBuilder

# Then run:
npm run dev

# If it says "Cannot find module", run:
npm install
```

### Page Won't Load

```
Check the URL:
http://localhost:5000/Resume.html  ← for resume
http://localhost:5000/cover-letter.html  ← for letter

(Make sure port is 5000, not 8000 or 3000)
```

### Download Doesn't Work

- Try the Print button instead (Ctrl+P, then Save as PDF)
- Check your Downloads folder (might already be there!)
- Try a different browser (Chrome recommended)

### Button Doesn't Respond

- Refresh the page (F5)
- Check browser console (F12) for errors
- Clear browser cache (Ctrl+Shift+Delete)
