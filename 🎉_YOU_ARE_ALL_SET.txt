📞 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

"Server won't start"
  → Check you're in right folder
  → Run: npm install
  → Run: npm run dev

"Page won't load"
  → Check URL has :5000 in it
  → Refresh page (F5)
  → Restart server

"Download button doesn't work"
  → Try Print button instead
  → Check Downloads folder (might already be there!)
  → Try different browser

"Button doesn't respond"
  → Refresh page (F5)
  → Press F12, check console for errors
  → Restart server
