================================================================================
                            IF SOMETHING FAILS
================================================================================

Can't install npm packages?
  → Check: Do you have Node.js installed? (node --version)
  → Try: npm cache clean --force
  → Try: Delete node_modules folder and package-lock.json, then npm install

Server won't start?
  → Check: Is MongoDB running?
  → Check: Is port 5000 available?
  → Check: Are there errors in the .env file?

Signup/Login doesn't work?
  → Check: F12 → Console for errors
  → Check: F12 → Network tab for API response
  → Check: Is backend running?

Resume won't load?
  → Check: F12 → Storage → LocalStorage for token
  → Try: Clear cache and login again
