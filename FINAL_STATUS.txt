📞 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────────

Q: Server won't start?
A: Run: npm install (then restart)

Q: Port 5000 in use?
A: Change PORT in .env file or close other apps

Q: Can't find module?
A: Run: npm install

Q: Pages show 404?
A: Check URL - use http://localhost:5000/ (not localhost:5000 without /)

Q: Styling not loading?
A: Clear cache: Ctrl+Shift+Delete (then Ctrl+F5 to refresh)

Q: API returning error?
A: After npm install, restart server
