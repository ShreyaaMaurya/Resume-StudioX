🚀 HOW TO USE
════════════════════════════════════════════════════════════════════════════

RESUME BUILDER:

1. Open: http://localhost:5000/Resume.html

2. Left Sidebar (5 sections):
   • ✨ AI BLUEPRINT ENGINE
   • ⬇️ EXPORT & DOWNLOAD ← Use this to download!
   • 01 / LAYOUT & IDENTITY
   • 02 / MANUAL COMPONENT MODULES
   • 03 / ACTIVE FIELDS PANEL

3. Click section buttons (📝, 💼, 🚀, etc.)

4. Edit content

5. Click: 📄 Download as PDF

6. Your resume.pdf is ready! ✅


COVER LETTER BUILDER:

1. Open: http://localhost:5000/cover-letter.html

2. Left Sidebar (5 sections):
   • ✨ AI LETTER OPTIMIZER
   • ⬇️ EXPORT & DOWNLOAD ← Use this to download!
   • 01 / YOUR INFO
   • 02 / RECIPIENT DETAILS
   • 03 / LETTER PARAGRAPHS

3. Fill in your info

4. Edit paragraphs

5. Click: 📄 Download as PDF

6. Your letter.pdf is ready! ✅


📄 DOWNLOAD WORKS TWO WAYS
════════════════════════════════════════════════════════════════════════════

Option 1: Direct PDF Download (Recommended)
  • Click: 📄 Download as PDF
  • Wait: 1-2 seconds
  • Result: File downloads to Downloads folder
  • Named: [YourName]_Resume.pdf
  • Quality: Professional PDF

Option 2: Browser Print (Fallback)
  • Click: 🖨️ Print
  • Print dialog opens
  • Select: "Save as PDF"
  • Choose: Location and name
  • Result: PDF saved where you choose

💡 QUICK TIPS
════════════════════════════════════════════════════════════════════════════

Resume Tips:
  • Use AI button to auto-generate layout
  • Keep sections focused and concise
  • Use bullet points for lists
  • Include metrics/achievements
  • Test print before sending

Letter Tips:
  • Keep intro short and engaging
  • Show specific company interest
  • Highlight relevant achievements
  • Keep closing clear and direct
  • Download both resume and letter

Download Tips:
  • Always download as PDF for sharing
  • Check filename has your name
  • Store PDFs in organized folder
  • Create versions for different jobs
  • Test print to verify formatting

Server won't start:

  → Restart server

  → Try Print button instead
  → Check Downloads folder
  → Try different browser

  → Refresh page (F5)
  → Open console (F12) - check for errors
  → Restart server
  → Try different browser
