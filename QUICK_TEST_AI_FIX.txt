TROUBLESHOOTING:

Problem: Still seeing template data after API key added
Fix:
  1. Ctrl+Shift+Delete to clear ALL cache
  2. Close browser completely
  3. Reopen http://localhost:5000
  4. Click Settings, verify key is there
  5. Try again

Problem: Getting error "Gemini HTTP 403"
Fix:
  1. API key might not have Multimodal Vision enabled
  2. Go to: https://console.cloud.google.com/
  3. Enable: "Generative Language API"
  4. Make sure key has vision permissions
  5. Or create new key with all permissions

Problem: Server won't start
Fix:
  1. Kill old process: Ctrl+C
  2. Wait 5 seconds
  3. Run: npm run dev
  4. If port still in use:
     netstat -ano | findstr :5000
     taskkill /PID <PID> /F
